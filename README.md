# Loan_Approvals
Build logistic regression, decision tree and neutral network models for auto-loan approvals

Lending Club (LC), which is a website that helps connect lenders with borrowers online. It also collects data on varied aspects of its lenders. My task is to help lenders formulate a strategy to mitigate losses when the borrowers are unable to pay off their loans. The classification goal is to predict if the client will default (yes/no).
